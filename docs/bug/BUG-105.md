## 描述

`apps/requirement_analysis/tests.py` 不存在。LVM 相关所有功能均无自动化测试覆盖：

- `AIModelService.preprocess_images()` — 无测试
- `UploadImageView` — 无测试
- LVM 预处理集成到生成管线 — 无测试
- 无 LVM 配置时的降级行为 — 无测试
- 图片格式/大小校验 — 无测试

## 风险

- 后续重构可能破坏 LVM 功能而不被察觉
- 降级路径正确性无法保证

## 修复建议

为目标添加 pytest/unittest 测试：
1. `preprocess_images` 单元测试（mock API 调用）
2. `UploadImageView` 接口测试（上传各种格式/大小）
3. 降级路径测试：无 LVM 配置时生成管线正常
